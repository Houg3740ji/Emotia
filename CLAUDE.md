# Instrucciones permanentes para Claude Code

## Git workflow

Después de cada conjunto de cambios significativos:

1. `git add .`
2. `git commit -m "descripción breve de los cambios"`
3. `git push origin main`

Al finalizar cada sesión de trabajo, siempre hacer commit y push de todos los cambios pendientes antes de responder al usuario.

## Idioma

Responder siempre en español.
